## Tính năng mới & Tối ưu hóa (v1.0.22)

*   **Khôi phục từ Cloud (Sync Down):** Hỗ trợ tính năng thiết yếu khi bạn đổi điện thoại hoặc cài mới. Ứng dụng giờ đây có thể khôi phục toàn bộ giao dịch, tài khoản cùng với danh sách File đính kèm từ Google Drive hoặc S3 Storage tương ứng.
*   **Phân nhóm dữ liệu Cloud:** Tự động sắp xếp gọn gàng các tệp đính kèm khi tải lên Drive/S3 thành 3 nhóm rõ ràng: `image/`, `media/` (Video, Audio) và `document/` (PDF, Excel...) để bạn dễ dàng quản lý bằng trình duyệt.
*   **Xóa toàn diện file đính kèm (Local + Cloud):** Mỗi khi bạn xóa một ảnh/đính kèm hoặc một phần giao dịch, hệ thống sẽ lưu vào hàng đợi và ra lệnh tiếp tục truy quét để xóa vĩnh viễn tệp rác đó trên Cloud (Drive/S3) nhằm tiết kiệm dung lượng.
*   **Sửa lỗi phân mảnh đường dẫn (Local):** Fix triệt để lỗi không nhận diện file đính kèm sau khi copy, đảm bảo file luôn tải và xem được mượt mà trên thiết bị nội bộ cục bộ ngay cả trước khi đồng bộ.
