## Tính năng (v1.0.25)

*   **Tinh chỉnh Tên Tệp đính kèm theo Transaction ID:** Lắng nghe phản hồi từ người dùng, việc tổ chức tên tệp trên Cloud đã được nâng cấp lên mức tối đa. 
Tất cả hình ảnh và file đính kèm giờ đây sẽ luôn ưu tiên **Gắn tiền tố mã Giao dịch (ID)** lên đầu và kết hợp với dữ liệu thời gian, tên nguyên bản. Cấu trúc hiện tại sẽ trông như sau:

`tx_{id_giao_dich}_{YYYYMMDD}_{Tên_tệp_gốc}_{Random4}.ext`

*(Ví dụ: Khi lưu 1 file vào giao dịch có id trong CSDL là 34 -> `tx_34_20260308_Hoadon_Dien_ab12.png`)*
Điều này cho phép bạn đối chiếu 1-1 dữ liệu hình ảnh với các Record Thu/Chi, Chuyển tiền sau khi Export dưới dạng JSON hoặc đọc trên Cloud Storage cực kì dễ dàng, chính xác.
